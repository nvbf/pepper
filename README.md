# pepper
Web-based overlay for norwegian volleyball.

# Prerequisit
A not to old versjon of node. 

# Install And Run

```bash 
export PORT=3000
yarn  #if you do not have yarn, try "npm install"
npm start
```

# Storybook 
`npm run storybook`

You will then find storybook on localhost:6006

# Paths
/  - Shows and example overlay screen
